FULL-STACK / AI / AUDIO GENERATION

AuraCast

A full-stack web application that transforms long-form eBooks into immersive, interactive, two-host AI podcasts.

  • Architecture: Parses expansive texts into digestible chapters for efficient backend processing.
  • Integration: Utilizes the Gemini 1.5 Pro API alongside Edge-TTS to synthesize dynamic audio conversations that users can listen to, interrupt, and query in real-time.

BACKEND / FULL-STACK / RENDER.COM

Ivy_AI: Full-Stack Implementation

A full-stack architecture separating sensitive logic from the user interface. This project utilizes a Python-based backend on Render to securely interface with LLM APIs.

  • Architecture: Built a specialized backend service on Render.com to manage API authentication and data security.
  • Security: Implemented server-side environment variables to protect sensitive API keys and validate user inputs.

AI / GENERATIVE ART / API

Muse & Machine

A generative art explorer that curates masterpieces from the Art Institute of Chicago and uses Gemini 2.0 Flash to synthesize real-time "AI Curator" insights.

  • Integration: Leveraged REST APIs to fetch high-resolution archival data.
  • LLM: Implemented Google Gemini to generate contextual historical analysis for each artwork.
